 🌾 Crop Monitoring Website

📌 Project Overview

The Crop Monitoring Website is a web-based application developed to support farmers, students, and agricultural researchers in understanding and monitoring crop-related data. The platform provides structured agricultural information and analytical insights that help improve decision-making and awareness in modern farming practices.

This project demonstrates the integration of web development and data analysis concepts to build a simple yet practical agricultural support system.

 🎯 Key Objectives

- Provide accessible crop information in a structured format  
- Support educational learning in agriculture and data science  
- Demonstrate practical implementation of web-based systems  
- Encourage technology adoption in farming  
- Present agricultural data in an easy-to-understand format  

🚀 Core Features

- User-friendly and clean web interface  
- Crop information display module  
- Data visualization support  
- Structured backend processing using Python  
- Educational support tool for students and beginners  
- Simple navigation and responsive design

🛠 Technologies Used
 Frontend:
- HTML5  
- CSS3  
- JavaScript  

 Backend:
- Python  

 Data & Analysis:
- Jupyter Notebook  
- Data processing libraries  
⚙️ Installation & Setup

To run this project locally:

1. Clone the repository  
git clone https://github.com/nivethithasenthilkumar/team_mini_project.git


2. Navigate to the project directory  
cd team_mini_project


3. Install required dependencies  
pip install -r requirements.txt


4. Run the application  
python app.py


If using Jupyter Notebook:
jupyter notebook
